# Run in Telegram WebApp

Enable dApps to use SDK in the Telegram webapp.

## Usage

- Insert script

```typescript
<script src="https://telegram.org/js/telegram-web-app.js" />
```

- Add configuration: Telegram botId

```typescript
import { ConfigProvider } from "@portkey/did-ui-react";

ConfigProvider.setGlobalConfig({
  socialLogin: {
    Telegram: {
      botId: TELEGRAM_BOT_ID,
    },
  },
  // ...
});
```

- Telegram initialisation: callback of logout (required)

```typescript
import { TelegramPlatform, did } from "@portkey/did-ui-react";

const handleLogout = () => {
  // logout
  did.logout();
};

TelegramPlatform.initializeTelegramWebApp({ handleLogout });
```

- Configure the bot token in Portkey service

| interface | environment | params |
| --- | --- | --- |
| **API**：api/app/telegramAuth/bot/register <br/> **method**： POST | TESTNET<br/>https://aa-portkey-test.portkey.finance<br/>MAINNET<br/>https://aa-portkey.portkey.finance | Body<br/><pre><code>{ "secret": "xxxxxx" }</code></pre> |
